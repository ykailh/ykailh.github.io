markdown常用语法

​ 工欲善其事必先利其器，markdown作为最适合程序员的文本编辑格式，是写技术博客不可或缺的一环，花了一天时间学习了相关的语法，总结分享作为新博客的第一篇文章。

文本格式

0.标题

​ 标题格式：# + 空格 + 标题内容，几个’#就代表几级标题。

1.字体

​ 字体格式：*文本内容* 或 _文本内容_ ，一个符号为斜体文本，两个符号为粗体文本，三个符号为**粗斜体文本**。

输入内容 效果
*斜体文本* 或 _斜体文本_ 斜体文本
**粗体文本** 或 __粗体文本__ 粗体文本
***粗斜体文本*** 或 ___粗斜体文本___ 粗斜体文本

2.删除线

​ 删除线格式：~~删除线文本~~

输入内容 效果
普通文本 普通文本
~~删除线文本~~ 删除线文本

3.下划线

​ 下划线格式：<u>带下划线的文本</u>

输入内容 效果
普通文本 普通文本
<u>带下划线的文本</u> 带下划线的文本

4.脚注

​ 使用脚注：文本内容[^脚注名称]，解释脚注：[^脚注名称]:脚注内容。

​ 效果：文本内容^脚注

5.分割线

​ 使用三个或以上 * 或 - 来使用下划线，可以在符号中加空格（可能会在编写软件中识别为其他符号，不建议使用），且本行中不可有其他内容。

分割线效果：

错误示范：***其他内容（未出现分割线效果）

列表与区块

1.无序列表

​ 无序列表格式：* 或 + 或 - 加上一个空格后面再加上内容，三个符号效果相同。

输入：

​ * 无序第一列

​ + 无序第二列

​ - 无序第三列

效果：

    * 无序第一列
  • 无序第二列
  • 无序第三列

2.有序列表

​ 有序列表格式：数字+ . + 空格 + 内容，即将无序列表的符号替换为数字加点的格式（例如1.）。

输入：

​ 1. 有序第一列

​ 2. 有序第二列

​ 3. 有序第三列

效果:

  1. 有序第一列
  2. 有序第二列
  3. 有序第三列

3.列表嵌套

​ 在语法中为在子列表前添加四个空格，不过一般都在编写软件上编写更推荐使用TAB（缩进）与SHIFT+TAB（反缩进）

​ 输入：

​ 1. 第一项：

​ * 第一项嵌套的第一个元素

​ * 第一项嵌套的第二个元素

​ 2. 第二项：

​ * 第二项嵌套的第一个元素

​ * 第二项嵌套的第二个元素

效果:

  1. 第一项：
    • 第一项嵌套的第一个元素
    • 第一项嵌套的第二个元素
  2. 第二项：
    * 第二项嵌套的第一个元素
   * 第二项嵌套的第二个元素

4.区块

​ 区块格式：在每行前加上 > 然后紧跟一个空格。

输入：

> 区块引用第一行

> 区块引用第二行

效果：

区块引用第一行

区块引用第二行

5.区块嵌套

​ 区块嵌套格式：每有一个 > 号就表示一层嵌套

输入：

> 最外层

>> 第一层嵌套

>>> 第二层嵌套

**效果: **

最外层

第一层嵌套

第二层嵌套

代码

1.代码片段

​ 代码片段格式：**` 代码内容 ` **(反引号)

​ 输入： ` int a ;` 效果： int a;

2.代码区块

​ 格式：用 ``` 包裹一段代码，并指定一种语言（也可以不指定）：

输入：

​ ``` java

public class Test {
public static void main(String[] args) {
System.out.println(“Hello World”);
}
}

​ ```

链接

1.直接显示链接

​ 语法格式: < 链接 > ** **效果: https://www.baidu.com/

2.用名称替代链接

​ 语法格式：**[链接名称](链接地址) **效果：这是我的博客

3.用名称和变量替代链接

​ 语法格式：[链接名称][变量名称]，[变量名称]: 链接。变量定义类似于脚注，效果与用名称替代类似，但方便多次使用。

效果：这是一个百度链接，再次使用百度链接。

表格

​ 用 | 来分隔单元格，- 来分隔表头。

输入：

| 表头 | 表头 |

| —- | —- |

| 单元格 | 单元格 |

| 单元格 | 单元格 |

效果：

表头 表头
单元格 单元格
单元格 单元格

默认为左对齐

将分隔符替换为-: , :- , :-: 分别表示右对齐，左对齐，居中对齐。

输入：

| 左对齐 | 右对齐 | 居中对齐 |

| :—- | —-: | :—-: |

| 单元格 | 单元格 | 单元格 |

| 单元格 | 单元格 | 单元格 |

效果：

左对齐 右对齐 居中对齐
单元格 单元格 单元格
单元格 单元格 单元格

图片

​ 图片插入格式：![图片名](图片相对路径/绝对路径/图片链接)；
